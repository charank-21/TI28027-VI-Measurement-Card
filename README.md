# TI28027 V&I Measurement Card

Four-layer PCB design of a TI28027 microcontroller based Voltage and Current (V&I) Measurement Card using Cadence OrCAD Capture CIS and Allegro PCB Designer.

## Project Overview

This project involves the schematic design, PCB layout development, and design file generation for a TI28027 microcontroller based V&I measurement card. The design was implemented as a four-layer PCB to support reliable signal routing and improved board organization.

## Tools Used

- Cadence OrCAD Capture CIS
- Cadence Allegro PCB Designer

## Project Files

- Schematic Diagram
- Gerber Output
- Bill of Materials (BOM)
- 3D PCB View
- Design Files (ZIP)

## Design Features

- Four-Layer PCB Architecture
- TI28027 Microcontroller Based Design
- Voltage and Current Measurement Interface
- Gerber Generation for PCB Fabrication
- Complete PCB Design Documentation
